# tailor
A modular AI framework that lets you stitch together models and tools into an assistant that fits you perfectly.

We will provide community and core plugins for common use cases.

The core plugins will contain things like memory, file access, web search, etc.

The community plugins will be provided by the community and will be available on a plugin marketplace.

You can also create your own plugins and share them with the community. By using a manifest file, you can describe your plugin and its capabilities.
